# E-commerce de Cartas TCG

Proyecto final del curso de ReactJS de CoderHouse. Es una tienda online dedicada a la venta de cartas coleccionables y accesorios para juegos de cartas (TCG).

## Sobre el Proyecto

Esta web permite a los usuarios navegar por un catálogo de productos, filtrar por categorías, ver detalles de cada carta o accesorio, agregar productos al carrito y finalizar la compra generando una orden en Firebase.

## Funcionalidades

- **Catálogo de productos**: Muestra todos los productos disponibles con sus imágenes, nombres y precios
- **Filtrado por categoría**: Navega por categorías específicas (cartas, accesorios, etc)
- **Detalle de producto**: Vista individual de cada producto con descripción completa
- **Carrito de compras**: Agregar, modificar cantidades y eliminar productos del carrito
- **Checkout**: Formulario de compra con validación de datos
- **Confirmación de compra**: Pantalla de confirmación con número de orden y resumen de productos comprados
- **Estados de carga**: Indicadores visuales mientras se cargan los datos

## Librerías Utilizadas

- **React**: Framework principal para la construcción de la interfaz
- **React Router DOM**: Manejo de rutas y navegación
- **Firebase/Firestore**: Base de datos para productos y órdenes
- **react-spinners**: Componentes de loading
